Architecture

BUILT
DIFFERENT

01
ZERO DEPENDENCIES

Node.js core only. No npm installs cascading into hundreds of packages. No CVEs you didn't write.

02
COLUMNAR STORAGE

Float64Array and TypedArrays throughout. Contiguous memory, V8 JIT-friendly, cache-efficient aggregations.

03
SHARED MEMORY

SharedArrayBuffer across Worker Threads. The entire dataset lives in one physical RAM block — no copies, no transfers.

04
PARALLEL BY DEFAULT

Worker Threads partition the dataset automatically. Every CPU core contributes. Results merge in the main thread.
